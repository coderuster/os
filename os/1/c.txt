cat: The cat command is used to concatenate files and print the output to the standard output.

Syntax:cat [options] [file(s)]

cat file1.txt file2.txt  # Concatenates file1.txt and file2.txt and prints the output
----------------------------------------------------------------
mkdir: The mkdir command is used to create directories.

Syntax:mkdir [options] directory_name

mkdir my_directory  # Creates a directory named "my_directory"
